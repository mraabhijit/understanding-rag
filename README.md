# Retrieval Augmented Generation

This repo hosts the users implementations of the course work from 
[Boots.dev](https://www.boot.dev/lessons/7a92d1c1-d202-481a-ae5f-14fc9f97b640)

The course dives into:
1. Preprocessing
2. TF-IDF
3. Keyword search
4. Semantic search
5. Chunking
6. Hybrid Search
7. LLMs
8. Reranking
9. Evaluation
10. Augmented Generation
11. Agentic
12. Multimodal

The course is meant to be simple and lightweight that explores buidling the intuition
of setting up a RAG system from scratch (almost).
